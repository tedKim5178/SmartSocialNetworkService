Smart SNS

This is a Smart SNS implementing pre-fetching mechanism. People can see high definition images more fast than any other SNS.

Intro

The size of image's have been getting bigger and bigger along with image's quality is getting better. Due to this, it takes long time when SNS user want to see the original high dimention image from SNS server. Because images that people want to see start to be sent from the server after people click the images. To make it better, Smart SNS came out to the world.

This looks just like ordinary SNS but actually it's not. Smart SNS implements pre-fetching function on background. So whenever user is not using network, pre-fetching function run automatically and download high dimention images that are likely to be seen by users from server.

Major functions

Prefetching

Prefetching function is started as soon as user sign in. It is shown in yellow bar above the screen for user to check what images is been downloading.

Registration

User can sign up to Smart SNS and sign in with their account to it. When user sign up, they will be forced to pick up three interests. These interests will be used for pre-fetching algorithm.

Timeline

User can see every posts uploaded by users. User can push like button on the post that he/she likes and also leave comments on it.

Post

User can upload post on their wall. User can pick up a photo from their phone gallery and write description about it. If user use # followed by some letters it will be hashtag automatically

My Timeline

User can see their timeline. It shows user's profile information on the top and images that the user posted in the past.

Tag Navigation

When user push hashtag on posts, tag navigation will lead user to tag search section. It shows user posts which contain the specific hashtag that user pressed
